Project 1: Prokudin-Gorskii Colorization

Automatic Color Channel Alignment

Implementation of an automatic colorization algorithm for digitized Prokudin-Gorskii glass plate images. Uses pyramid-based alignment and edge detection to align red and green channels with blue reference, creating color images from historical black-and-white glass plates.
